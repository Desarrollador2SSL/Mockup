Menú con Siglas Integradas

Instrucciones

1. Haz clic en el botón circular azul para colapsar/expandir el menú lateral.

2. Cuando el menú esté colapsado, abre cualquier sección con submenús.

3. Los submenús ahora se muestran dentro de la misma barra lateral con las siguientes siglas:

  • SP - Solicitudes de Pedido
  • CO - Cotizaciones
  • OC - Órdenes de Compra
  • CEO - Control Entrega por OC
  • CED - Control Entrega por Despacho
  • EP - Estado de Pagos
  • AOP - Administración de Obras/Proyectos
  • AU - Administración de Usuarios

4. Pasa el cursor sobre las siglas para ver un tooltip con el nombre completo.

Características del Nuevo Diseño

✓ Los submenús permanecen dentro de la barra lateral cuando está colapsada

✓ Las siglas aparecen en cajas redondeadas con efecto hover

✓ Transición suave entre texto completo y siglas

✓ Tooltips informativos al pasar sobre las siglas

✓ Diseño consistente y limpio

✓ En móviles se mantiene el texto completo para mejor legibilidad

Ventajas de este Diseño

• Mayor claridad: Los submenús siguen la misma estructura visual del menú principal

• Mejor uso del espacio: No hay elementos flotantes que puedan solaparse

• Experiencia consistente: El comportamiento es predecible y fácil de entender

• Accesibilidad mejorada: Las siglas son claras y los tooltips ayudan a identificar cada opción
