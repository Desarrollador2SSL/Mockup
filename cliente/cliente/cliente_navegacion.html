Visión General del Sistema

Conozca cómo SSL v4.0 optimiza su proceso de compra y gestión de materiales

¿Qué es SSL v4.0?

SSL v4.0 es una plataforma integral que conecta a clientes con proveedores para la gestión eficiente de materiales de construcción. El sistema automatiza todo el proceso desde la solicitud hasta la entrega y pago.

1

Solicitar

Cree solicitudes de materiales para sus obras de manera rápida y organizada

2

Cotizar

Reciba y compare múltiples cotizaciones de diferentes proveedores

3

Comprar

Genere órdenes de compra con los mejores precios y condiciones

4

Recibir

Controle las entregas y valide la recepción de materiales

5

Pagar

Gestione los pagos y mantenga control de su cuenta corriente

¡Importante!

SSL actúa como intermediario entre su empresa y múltiples proveedores, centralizando la gestión y optimizando los procesos de compra.

Etapas del Proceso

Primera Parte: Solicitud y Cotización

1. Generar Solicitud de Pedido

El cliente crea una solicitud especificando:

  • Obra asociada
  • Productos requeridos
  • Cantidades necesarias
  • Fechas de entrega
2. SSL Revisa y Aclara

SSL procesa la solicitud:

  • Verifica productos
  • Aclara especificaciones
  • Puede generar nueva versión
3. Solicitar Cotizaciones

SSL contacta proveedores:

  • Envía solicitudes a múltiples proveedores
  • Recibe cotizaciones
  • Consolida información
4. Cuadro Comparativo

Cliente analiza opciones:

  • Compara precios
  • Evalúa condiciones
  • Selecciona productos/proveedores
  • Chat para aclarar dudas
5. Generar Orden de Compra

Confirmación de compra:

  • Cliente acepta cotización
  • Genera una o más OC
  • Define condiciones finales

Segunda Parte: Compra y Entrega

1. Crear OC en Sistema

Cliente formaliza la orden:

  • Sube OC al sistema
  • Define fechas de entrega
  • Especifica condiciones
2. SSL Acepta/Rechaza

Validación de la orden:

  • Revisa términos
  • Confirma disponibilidad
  • Acepta o solicita cambios
3. Planificación de Entregas

SSL coordina logística:

  • Programa con proveedores
  • Considera peso/volumen
  • Gestiona documentación
  • Tramita aduana si aplica
4. Documentación

Generación de documentos:

  • Declaración Ingreso (DI)
  • Declaración Salida (DS)
  • Listado Productos Entregados (LPE)
5. Recepción y Control

Validación de entregas:

  • Recepción en obra
  • Verificación de productos
  • Reporte de incidencias
  • Control de cumplimiento

Tercera Parte: Facturación y Pago

1. Generación de Cobranza

Por cada entrega (LPE):

  • Se genera factura
  • Se calcula monto a pagar
  • Se establece vencimiento
2. Proceso de Pago

Cliente gestiona pagos:

  • Revisa facturas pendientes
  • Realiza transferencia
  • Informa pago en sistema
3. Documentación de Pago

Respaldo del proceso:

  • Sube comprobante
  • Asocia a facturas
  • Mantiene trazabilidad

Flujo de Documentos - Opción Cotización Separada vs Unificada

Cotización Separada

Se generan tantas cotizaciones como proveedores seleccionados

  • Ventaja: Mejor control por proveedor
  • Ventaja: Trazabilidad específica
  • Desventaja: Más documentos por administrar

Cotización Unificada

Una sola cotización independiente de proveedores

  • Ventaja: Simplicidad administrativa
  • Ventaja: Una sola OC
  • Desventaja: Menor trazabilidad por proveedor
